Travel Recommendation PoC

Find the right destination through conversation, not filter menus.

This demo uses an LLM to interpret travel intent and GraphQL to retrieve structured destination matches.

How it works

  • Understand traveler intent
  • Translate it into filters
  • Query structured destination data
  • Explain the best matches clearly
